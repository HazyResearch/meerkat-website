Foundation Models meet Data and Apps.

Meerkat is a Python library to wrangle unstructured and structured data using dataframes, connect your data to foundation models, and build full scale web apps on top (or just run interfaces in a notebook!).

Install Meerkat

pip install meerkat-ml
